# Hit Init

Single-page HTML baseball arcade game. The 1989 IP's black-cat batter steps up — horizontal/vertical hit-gauge mini-game → 3D perspective alley where the ball rolls into one of five holes → real-baseball base advancement → 3-out game over.

## Play

Open `index.html` in any modern browser. No build, no install, no external dependencies.

**Controls**: SPACE / TAP — stop the gauge bar. R — steal (when available).

## Project notes

- Single HTML file. All HTML/CSS/JS/Canvas inline.
- Master illustration `game.png` (1024×1536) sliced into `assets/v2_title.png`, `v2_game.png`, `v2_roll.png` for the three screens.
- HUD: 32 painted panels (8 base states × 4 out counts) swapped by filename.
- See `CLAUDE.md` for project conventions and `HANDOFF_v1.2.md` for the latest handover document.
